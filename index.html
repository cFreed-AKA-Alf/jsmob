Jsmob

A JS add-on to instantly make an existing site become responsive

Download .zip Download .tar.gz View on GitHub

What is it?

The jsmob project is an attempt to have a simple and easy to use javascript module which can be added to any existing site to instantly make it become responsive with a minimum effort.
There is no attempt (at least for now) to minimize the loading time (like selecting between different versions of images or different dedicated CSS files depending on the device window size, and so on).

In contrast, "simple and easy to use" means that for jsmob to work on a site you must only:

  • hook the jsmob.js file
  • add a few number of class names to define the desired behaviour
  • and sometimes adapt the columns HTML structure, and therefore turn some #ident CSS specifications into .ident.

What does it look like?

You may examine a test case in operation: either observe it in different window sizes on a desktop, or using a mobile device.
You will notice that it allows to have different breakpoints for different parts of the page, and it adapts instantly when switching between portrait and landscape.

When a page part passes its defined breakpoint, depending on its own settings:

  • adjacent columns become stacked (in a different order than in full layout, if so defined)
  • a hover-sensitive menu becomes touch-sensitive (and is replaced by a "commander" button, if marked as main menu)
  • elements marked as optional disappear from the page
  • and some other little touchy behaviours

How to use it on your own test case?

Look at the readme.md file in the jsmob project repository for detailed information.
